## Project Definition

SciReagent is:

> An AI-Native Scientific Knowledge Graph Platform for Nucleotides, Click Chemistry, and Related Scientific Reagents.

SciReagent is NOT:

* Shopify Store
* Generic Laboratory Catalog
* Sigma-Aldrich Clone
* MedChemExpress Clone
* Traditional Product Database

The system is built around:

```text
Research Goal → Application → Method → Protocol → Product → SKU
```

NOT:

```text
Category → Product → SKU
```
