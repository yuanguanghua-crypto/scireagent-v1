## Fix 1: ORM Query Path

**Wrong:**
```python
products_count=Count('methods__productmethod__product', distinct=True)
```

**Correct:**
```python
products_count=Count('methods__product_methods__product', distinct=True)
```

**Reason:** `ProductMethod.method` has `related_name='product_methods'`, not `productmethod`.

## Fix 2: Related Products Serialization

**Wrong (in View):**
```python
'products': ProductBriefSerializer(related, many=True).data,
```

**Correct:**
```python
'products': RelatedProductSerializer(related, many=True).data,
```

**Reason:** `get_related_products()` returns `list[dict]`, not QuerySet. `ProductBriefSerializer` is a `ModelSerializer` that expects model instances. `RelatedProductSerializer` is a plain `Serializer` that handles dicts.

## Fix 3: Missing Serializer Definitions

**Add to Section 1.1:**

```python
class ReferenceBriefSerializer(serializers.ModelSerializer):
    class Meta:
        model = Reference
        fields = ['id', 'title', 'journal', 'year', 'doi']

class ProductFullSerializer(serializers.ModelSerializer):
    skus = SKUSerializer(many=True, read_only=True)
    documents = ProductDocumentSerializer(many=True, read_only=True)
    product_class_name = serializers.SerializerMethodField()
    product_class_path = serializers.SerializerMethodField()

    class Meta:
        model = Product
        fields = [
            'id', 'name', 'slug', 'catalog_no', 'cas', 'smiles', 'synonyms', 'inchi',
            'formula', 'molecular_weight', 'purity', 'concentration', 'storage',
            'shipping', 'lead_time', 'handling_notes', 'shelf_life', 'research_use_only',
            'overview', 'structure_svg', 'seo_title', 'seo_description',
            'category_l1', 'category_l2', 'status', 'product_class_id',
            'product_class_name', 'product_class_path',
            'skus', 'documents', 'created_at', 'updated_at',
        ]

    def get_product_class_name(self, obj):
        return obj.product_class.name if obj.product_class else None

    def get_product_class_path(self, obj):
        if not obj.product_class:
            return []
        path = []
        pc = obj.product_class
        while pc:
            path.insert(0, pc.name)
            pc = pc.parent
        return path
```

- `methods_count` (not `method_count`)
