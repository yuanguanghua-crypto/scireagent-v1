# 07 Decision Framework

## Priority Order

```text
1. Knowledge Graph Integrity
2. Research User Experience
3. SEO and AEO
4. Commercial Conversion
5. Code Elegance
6. Development Convenience
```

If conflict occurs: higher priority wins.

---

## Examples

**Beautiful UI vs Knowledge Graph visibility**
→ Choose Knowledge Graph.

**Less code vs Research Goal entry**
→ Choose Research Goal.

**Faster development vs SEO requirements**
→ Choose SEO.

**Conversion optimization vs scientific context**
→ Choose scientific context.
