# 04 Homepage Acceptance Criteria

## Visual Weight

| Section            | Weight |
| ------------------ | ------ |
| Hero Search        | 20%    |
| Research Goals     | 15%    |
| Featured Solutions | 15%    |
| Featured Products  | 25%    |
| Knowledge Graph    | 15%    |
| Publications       | 5%     |
| Custom Synthesis   | 5%     |

Deviation >10% requires approval.

---

## Hero Search Requirements

Must contain:
- Headline
- Search Box
- Suggested Queries
- CTA Buttons
- Knowledge Graph Illustration

---

## Featured Products

Requirements:
- 6–8 products
- Desktop: 4 columns
- Tablet: 2 columns
- Mobile: 1 column

Card:
- Structure Image
- Product Name
- CAS
- Purity
- Application Tags
- Price
- View Details
- Add to Cart
- Request Quote

---

## Knowledge Graph Preview

Must support:
- Zoom
- Click
- Hover
- Navigation

Must display:
```text
Application → Method → Protocol → Product → SKU
```

---

## Performance

- Lighthouse: > 90
- CLS: < 0.1
- LCP: < 2.5s
