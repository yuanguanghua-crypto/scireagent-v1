## 注意事项

1. **多个值用分号 `;` 分隔**
2. **不确定的可以留空**，后续我们补充
3. **可以新增选项**——如果上面的清单没有涵盖，直接写上即可
4. **同一类产品可以复制粘贴**——比如所有 2'-Azido 系列的填写基本一致
5. **参考文献填 PMID 号即可**——不知道 PMID 可以填 DOI 或论文标题
